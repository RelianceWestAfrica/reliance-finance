# Premier login et activation du compte

## Vous avez recu une invitation par email

L'email d'invitation contient un **lien magique** valable 24h. Cliquez
dessus depuis le poste sur lequel vous voulez activer votre compte.

1. Cliquez sur **"Activer mon compte"** dans l'email
2. Vous arrivez sur la page **"Definir mon mot de passe"**
3. Choisissez un mot de passe respectant les regles :
   - 12 caracteres minimum
   - 1 majuscule, 1 minuscule, 1 chiffre, 1 caractere special
   - Pas reutilise d'un de vos comptes externes
4. Confirmez le mot de passe
5. Vous etes redirige vers le **tableau de bord**

A partir de la, vous vous connectez avec votre email + mot de passe sur
https://finances.rwa-core.com/login.

## Vous n'avez pas recu d'invitation

Contactez votre **Administrateur** ou votre **DAF Pays** pour qu'il vous
cree un compte. Verifiez aussi le dossier spam.

## Premier login - admin systeme uniquement

Si vous etes le **premier administrateur** (compte cree par le seed), vos
credentials par defaut sont :

- Email : `admin@reliancewestafrica.com`
- Mot de passe initial : `ChangeMe123!`

**Changez ce mot de passe immediatement** apres la premiere connexion via
**Profil > Securite**.

## Profil utilisateur

Apres le premier login, configurez votre profil :

1. Cliquez sur votre nom en haut a droite > **Mon profil**
2. Renseignez votre **nom affiche** (sera visible dans les signatures
   et l'audit log)
3. Selectionnez votre **timezone** (Africa/Lome par defaut, mais ajustable
   si vous travaillez en dehors)
4. Selectionnez votre **locale** (fr-FR par defaut)
5. Enregistrez

## Securite du compte

- **Ne partagez jamais** votre mot de passe
- Verrouillez votre poste si vous vous absentez (les sessions sont coupees
  apres 15 min d'inactivite mais c'est insuffisant si quelqu'un acces
  physique pendant ces 15 min)
- En cas de doute, changez votre mot de passe **immediatement** depuis
  **Mon profil > Securite**

## Que faire en cas de blocage ?

| Probleme | Action |
| -------- | ------ |
| Mot de passe oublie | Lien "Mot de passe oublie" sur la page login -> email magique |
| Compte bloque (5 tentatives loupees) | Contactez l'Administrateur (deblocage manuel) |
| Email d'invitation expire (24h) | Demandez une nouvelle invitation a l'Administrateur |
| Mon role n'est pas le bon | Contactez votre DAF Pays ou DFG (uniquement eux peuvent modifier les memberships) |
